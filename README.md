# EEG Source Localization & Mediation Analysis Pipeline

A complete EEG data processing pipeline based on MNE-Python, covering all steps from raw `.set` files to sLORETA source localization, regional power extraction, bootstrap mediation analysis, and publication-ready visualization.

---

##  Project Structure

```
SHN/
├── config.py                          # Central configuration (paths, parameters, channel names, ROI coordinates)
├── utils.py                           # Utility functions (scent classification, log power transformation)
├── step1_generate_asc.py              # Step 1: Generate ASC files
├── step2_source_localization.py       # Step 2: sLORETA source localization → NIfTI
├── step3_extract_power.py             # Step 3: Regional power extraction
├── step4_mediation_analysis.py        # Step 4: Bootstrap mediation analysis + visualization
├── step5_interaction.py      # Step 5: Brain region × scent interaction analysis
├── step6_lmm_statistics.py            # Step 6: Multi-band LMM interaction statistics
├── step7_interaction_visualization.py # Step 7: Interaction highlight regression plots
├── requirements.txt                   # Dependencies
└── README.md                          # This file
```

---

##  Execution Flow

```
Step 1 ──→ Step 2 ──→ Step 3 ──┬──→ Step 4 (Mediation analysis)
                                ├──→ Step 5 (Interaction analysis)
                                ├──→ Step 6 (LMM statistics)
                                └──→ Step 7 (Interaction visualization)
```

> ️ **Steps 1 → 2 → 3 must be executed in order** (each step depends on the output of the previous step).  
>  **Steps 4 ~ 7 can be run independently** as long as the Step 3 Excel output is ready.

---

##  Detailed Step Descriptions

### Step 1 — Generate ASC Files
```bash
python step1_generate_asc.py
```
| Item | Description |
|------|-------------|
| **Input** | EEGLAB `.set` file |
| **Processing** | Load raw EEG → Resample to 125 Hz (with anti-aliasing filter) → Slice by time segments |
| **Output** | One `.asc` plain text matrix file per segment (`%.12e` scientific notation to prevent µV-level truncation) |
| **Config** | `SET_FILE`, `RESAMPLE_RATE`, `SEGMENTS`, `ASC_OUTPUT_DIR` |

---

### Step 2 — sLORETA Source Localization → NIfTI
```bash
python step2_source_localization.py
```
| Item | Description |
|------|-------------|
| **Input** | `.asc` files generated by Step 1 |
| **Processing** | Bandpass filtering per frequency band → sLORETA inverse operator → Hilbert envelope power → Voxel-level power maps |
| **Output** | One `.nii.gz` file per segment × per frequency band |
| **Caching** | Source space (SRC), forward model (FWD), and inverse operator (INV) are cached after first computation in `MNE_CACHE_DIR` and loaded in subsequent runs |
| **Config** | `ASC_OUTPUT_DIR`, `NIFTI_OUTPUT_DIR`, `MNE_CACHE_DIR`, `SUBJECTS_DIR`, `SUBJECT`, `CH_NAMES`, `FREQ_BANDS` |

**Frequency Bands:**

| Band | Frequency Range |
|------|-----------------|
| Delta | 0.5 – 4 Hz |
| Theta | 4 – 8 Hz |
| Alpha | 8 – 15 Hz |
| Beta | 15 – 25 Hz |
| Gamma | 25 – 50 Hz |

---

### Step 3 — Regional Power Extraction
```bash
python step3_extract_power.py
```
| Item | Description |
|------|-------------|
| **Input** | `.nii.gz` files from Step 2 + metadata Excel |
| **Processing** | Extract mean power from 9 brain regions (spherical ROIs, radius 15mm) using `NiftiSpheresMasker` |
| **Output** | Append columns `M_LF_Power` ~ `M_RP_Power` (9 columns total) to Excel |
| **Crash Prevention** | `gc.collect()` after each trial to release memory; auto-save every 10 trials |

**9 Brain Regions MNI Coordinates:**

| Abbr | Region | MNI (x, y, z) |
|------|--------|----------------|
| LF | Left Frontal | (-40, 35, 30) |
| MF | Middle Frontal | (0, 45, 20) |
| RF | Right Frontal | (40, 35, 30) |
| LT | Left Temporal | (-55, -10, 10) |
| MC | Central | (0, -10, 50) |
| RT | Right Temporal | (55, -10, 10) |
| LP | Left Parietal | (-40, -60, 40) |
| MP | Middle Parietal | (0, -60, 30) |
| RP | Right Parietal | (40, -60, 40) |

---

### Step 4 — Bootstrap Mediation Analysis + Visualization
```bash
python step4_mediation_analysis.py
```
| Item | Description |
|------|-------------|
| **Input** | Step 3 Excel (with 9 regional power columns) |
| **Processing** | Scent dummy coding → Log10 transformation → 5000-iteration bootstrap mediation (pingouin) |
| **Output** | ① Full results Excel ② Significant pathways Excel ③ Composite figure (heatmaps A/B + forest plot C) |
| **Significance** | 95% CI excluding 0 → significant |

---

### Step 5 — Brain Region × Scent Interaction Analysis
```bash
python step5_interaction.py
```
| Item | Description |
|------|-------------|
| **Test** | **Brain region × scent category interaction**: For each ROI, fit `Y ~ Power * Scent + (1|Subject)` |
| **Output** | Console output with significance results, Excel file with detailed interaction statistics |
| **Significance** | p < 0.05 for interaction term |

---

### Step 6 — Multi-band LMM Statistics Table
```bash
python step6_lmm_statistics.py
```
| Item | Description |
|------|-------------|
| **Input** | Excel files from multiple frequency bands (Delta / Beta / Gamma) |
| **Processing** | For each band × each target ROI, extract β coefficients, t-values, and p-values for LMM interaction terms |
| **Output** | APA-format Excel table (only p < 0.10 terms retained), ready for manuscript inclusion |
| **Significance Markers** | `***` p<0.001 · `**` p<0.01 · `*` p<0.05 · `†` p<0.10 |

---

### Step 7 — Interaction Highlight Visualization
```bash
python step7_interaction_visualization.py
```
| Item | Description |
|------|-------------|
| **Input** | Step 3 Excel |
| **Processing** | Facet by brain region with regression lines; highlight specified scent category (bold red), fade other categories (gray) |
| **Output** | 600 DPI high-quality PNG image |

## ️ Configuration

**All paths and parameters are managed centrally in `config.py`.** When switching machines or datasets, simply modify this one file:

```python
# Example key configuration items in config.py

SET_FILE = r"path/to/your/set/file.set"           # Step 1 input
ASC_OUTPUT_DIR = r"ASC/output/directory"          # Step 1 output / Step 2 input
NIFTI_OUTPUT_DIR = r"NIfTI/output/directory"      # Step 2 output
NIFTI_INPUT_FOLDER = r"NIfTI/input/directory"     # Step 3 input
EXCEL_INPUT_PATH = r"metadata/excel/path.xlsx"    # Step 3 input
SUBJECTS_DIR = r"FreeSurfer/directory"            # Path to fsaverage
```

##  Console Output Conventions

All scripts use consistent prefixes for easy log scanning:

| Prefix | Meaning |
|--------|---------|
| `[INFO]` | General progress information |
| `[OK]` | Individual task completed successfully |
| `[DONE]` | Entire process completed |
| `[WARNING]` | Non-critical warnings |
| `[ERROR]` | Processing exceptions |
| `[MISSING]` | Missing files |
| `[SAVE]` | Auto-save progress |
| `[TEST]` | Statistical test started |
| `[RESULT]` | Statistical test conclusion |

## Data availability

Raw EEG recordings and subject-level experimental data are not included in this repository due to privacy and ethical restrictions. The repository contains only the Python scripts required to reproduce the computational pipeline when appropriate input data and metadata are provided.

## Relation to the thesis

This repository contains the Python implementation of the computational pipeline described in the master's thesis:

"Mediation analysis and neural network models for predicting group response to aromas using 3D reconstruction of brain activity."

The pipeline includes EEG preprocessing, sLORETA-based source reconstruction, ROI feature extraction, bootstrap mediation analysis and linear mixed models.
